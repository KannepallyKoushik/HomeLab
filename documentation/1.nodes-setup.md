## Setup Steps

### 1. Hardware & Virtualization Setup

- **Control Plane:** VM on Mac using VMware Fusion.
- **Worker Nodes:** Dedicated physical machines with Ubuntu Server.
- **Virtualization Node:** Proxmox VE installed on k-pve-slim for VM orchestration and future Talos Linux Kubernetes clusters.
- **Future Plan:** Expand Proxmox installation to all nodes for advanced VM-based service segregation and multi-cluster management.

### 2. OS Installation

#### Control Plane - Ubuntu VM Setup (VMware Fusion)
1. Create a new VM in VMware Fusion (Linux → Ubuntu 64-bit).
2. Allocate resources: 3 cores, 4GB RAM, 30GB disk, bridged networking.
3. Install Ubuntu Server 24.04 LTS and configure static IP (192.168.0.110).

#### Worker Nodes - Ubuntu Server Installation (Physical)
1. Create bootable USB with Ubuntu Server 24.04 LTS.
2. Install on hardware, set static IPs (192.168.0.200, 192.168.0.150).
3. Configure SSH, update system, and verify connectivity.

#### Virtualization Node - Proxmox VE Installation
1. Download and install Proxmox VE ISO on k-pve-slim.
2. Configure Proxmox for local network access and VM creation.

### 3. Network Configuration
- Assign static IPs to all nodes.
- Set up SSH and key-based authentication.
- Configure DNS (see [local-dns-server-setup.md](./3.local-dns-server-setup.md)).

### 4. Service Deployment
- Install and configure K3s cluster ([k3s-cluster-setup.md](./2.k3s-cluster-setup.md)).
